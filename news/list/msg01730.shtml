In one word, WML-2.0.2 is out.  I am working very hard those days and
prefer going to sleep than writing detailed information on changes,
sorry.

This version implements the syntax for tag processing discussed here,
and should be easier to use than wml-2.0.0 and wml-2.0.1, because
compatibility with WML 1.x has been improved.

I am really happy with the latest mp4h-1.0.2 which is the processor run
on Pass 2, thanks to those who contribute to make it much better :o)
