csmall@debian.org writes:
> | WML doesn't create HTML 4.0 standard code.

> | The main problem was it doesn't like the name field in 
> | a <IMG> tag.

This bug has been fixed by the W3C a week after it was reported. As of
HTML 4.01 (http://www.w3.org/TR/html40/) img has a name attribute.

Change your DOCTYPE daclarations to one of:
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
            "http://www.w3.org/TR/html40/loose.dtd">
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
            "http://www.w3.org/TR/html40/strict.dtd">
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
            "http://www.w3.org/TR/html40/frameset.dtd">

Yours, Florian.
