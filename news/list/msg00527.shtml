Meta-HTML tags are case sensitive ...

  • From: Fritz Zaucker <nospam@thanx>
  • Date: 30 Mar 1999 14:25:38 +0200
... and I guess they shouldn't be as HTML is not case sensitive either.

The problem shows up if you redefine an existing HTML tag, say

<define-container H1>
whatever
</define-container>

then 

<h1> is NOT redefined.

The solution of redefining both <h1> and <H1> gets a bit more
cumbersome with tags with longer names, as all combinations of upper
and lower case would have to be redefined.
