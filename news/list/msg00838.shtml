Bug report: mistake in documentation of wml:.std.info

  • From: Uwe Mindrup <nospam@thanx>
  • Date: Thu, 8 Jul 1999 09:51:12 +0200
Hi,

the documentation to wml::std::info describes a variable "keyword" like
in 
  <info style=meta keyword="foo bar baz">
but this don't work. Only the variable "keywords" is recognized:
  <info style=meta keywords="foo bar baz"> 
works as expected.

It is only a little mistake in the documentation.
