HyperText Markup Language (HTML)

These are references to the HyperText Markup Language, the current main definition language for webpages. The current version is 4.0, which is not yet supported in major browsers. The previous version 3.2 has good support in Netscape Communicator 3 and 4 and in Microsoft Internet Explorer 4.
