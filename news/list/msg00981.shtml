In an earlier message I wrote about how to check if a certain
function was already defined. 

I am still interested in that information. I wanted it to check if a
certain function was defined, and so do a certain action (using the
<if> tag).

I wated it to check what languages were defined using the lang
extension. (Pretty strange request, you may think).

Well, I discovered this extension uses a variable `__languages[]' to
store the languages signs. So if I want to check if a language is
being used I simply need to use this var and not care about the tags.
