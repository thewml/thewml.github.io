We use imagemagick instead of gFont. And yes, there are perl-bindings
for it, so you can call it from WML.

Look at 

http://www.ee.ethz.ch/isg or http://www.ee.ethz.ch/student-computing 
or http://opensource.ee.ethz.ch 

for some examples.

All sites have the corresponding .wml for the .html files online at
the same location. 

The last site just uses #include for the template, so there it should
be easy to find them.

The other two sited "hide" the location of the template-files (by
having a -P prepreocessor), but the directory structure is identical
to the last site.

There magik.inc files are doing the image-magik thing ...
