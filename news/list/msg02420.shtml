Hello,

When pass 7 (Markup Code Fixup) detects a series of 6 digits in a tag
parameter, it adds a # before it. It seems to treat the parameter as a
color reference, even if it is not called COLOR?

Input:

<protect pass=7><A NAME="123456"></A></protect>
<A NAME="123456"></A>
<FOO bar="123456">
<FOO bar="1234567">

Output:

<A NAME="123456"></A>
<A NAME="#123456"></A>
<FOO bar="#123456">
<FOO bar="1234567">
