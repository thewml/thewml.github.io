wml simply ignores the php tags, so you cannot use php for wml macros but
you can use wml to generate headers, footers and navigation bars for php
pages, good for mixing static and dynamic pages.

One possible problem:

Your php code must produce proper html and quote all parameters, else the
tidy part of wml run will create additional "" pairs that break your php
strings.
