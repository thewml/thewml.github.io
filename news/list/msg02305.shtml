xtable bug

  • From: Warren Turkal <nospam@thanx>
  • Date: Wed, 18 Jul 2001 22:42:39 -0400
Code to reproduce error:
<xtable>
(1,1)
  Hello
(10,1)
  , World!
</xtable>

What it does:
In table position (1,1), "Hello" is printed, which is correct.  In 
table position (10,1), "Hello, World!" is generated, which is incorrect.

What it should do:
It should only generate ", World!" in table position (10,1).
