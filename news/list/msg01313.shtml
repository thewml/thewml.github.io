  • From: Franz Knipp <nospam@thanx>

Hello,

these are my proposed enhancement of the include tag. I've documented my
changes in the documentation part.

You can apply the attached file with:

[user@host wml-1.7.4/wml_backend/p1_ipp]$patch ipp.src diff.ipp.src 

This will introduce new parameters for the #include '' directive, namely
INCLUDE_SORT, INCLUDE_REVERSE, INCLUDE_MAX=n - see the file for more
information.

I'm thinking of another new parameter like INCLUDE_RANGE but I'm not sure
if this is useful.
