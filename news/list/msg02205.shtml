On Wed, Apr 18, 2001 at 09:13:43AM -0400, James A. Treacy wrote:
> Is there a way to include all the files in a directory, like
> #include 'blah/*'
> that will include the files in alphabetic order?

Hi Jay,

try
  #include 'blah/*' IPP_SORT
with wml > 1.7.4

This is described in wml_p1_ipp, and is a contribution by Franz Knipp.
