I finally made it. It was a compiler problem ... The upgrade to SuSE
6.0 installed the egcs as default compiler (therefore I could not 
even compile a new Kernel ...) I had to install the gccfrontend-
package to get the gcc back as default compiler ...

Now everything works fine (configure (even withput the setpgrp-patch 
...), make and make test ....)

express, that WML ist THE BEST tool I ever found for Website-design.
