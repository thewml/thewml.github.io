Website META Language, Version 1.6.7

 ftp://ftp.engelschall.com/sw/wml/

The corresponding entry from the ChangeLog file:

  Changes between 1.6.6 and 1.6.7:
      970626 - Added missing link to wml::des::preload to WMd [Thanks to
               Alvise Belotti <alvise@lettere.unipd.it> for hint]
      970628 - Added "target" attribute to <logo> tag from wml::std::logo
               [Thanks to Florian Kuehnert <sutok@gmx.de> for patch]
      970629 - Upgraded txt2html aux tool from version 1.24 to 1.25
      970705 - Fixed to bugs in wml::des::gfont [Thanks to Jordan Baker
               <jbb@interlog.com> for hints]
      970726 - Make divertion example in wml_tutorial more robust and less
               confusing [Thanks to Florian Hars <hars@math.uni-hamburg.de>
               for hint]
      970729 - Add "powerd by VIM" logo for wml::std::logo
      970730 - Fixed synopsis of htmlfix manual page 
               [Thanks to Kurt D. Zeilenga <Kurt@boolean.net> for hint]
             - Fixed handling of EOL-comments (#...), now ``\#xxx''
               inside style sheets works fine again.
      970801 - Fixed <sc>..</sc> tags: They were confused when
               in the processed body other tags occured.  [Thanks to Kurt D.
               Zeilenga" <Kurt@Boolean.Net> for hint]
      970802 - Upgraded IO Perl module from version 1.19 to 1.20
             - Replaced mkinstalldirs and install.sh scripts with
               cleaned variants I've done for Apache 1.3's APACI
             - Replaced build user/host/domain/time determination
               stuff with a new `buildinfo' script I've written
               for Apache which is more robust.
             - Upgraded ePerl from version 2.2.12 to 2.2.14. This
               especially provided Perl 5.005 support ;-)
