Cascading Style Sheets (CSS)

Cascading Style Sheets are the official W3C standard for specifying layouting attributes for webpages. The current version Level 1 (CSS1) is supported in Netscape Communicator 4 and Internet Explorer 4. The next generation is Level 2 (CSS2), currently only defined as a working draft with no real support in browsers.
