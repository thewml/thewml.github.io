On Fri, 5 Mar 1999, Frederic Le Mouel wrote:

> Hi,
> 
> When I pass height and width to rollover tag, height and width
> are well-passed to html img tag, but when I use lang tag in 
> rollover tag, height and width arguments don't passed.

In fact, height and width are not valid attributes for rollover, and are
removed. But wml_p7_htmlfix put them back into the HTML code.

With lang attributes, wml_p7_htmlfix can not guess the image size,
because it knows only the starred filename, and has no idea on what its
expansion will be.
