> Now, my small problem with include files. In a subdirectory

> root directory, it doesn't find the include file.
> 
> What should be the include statement in the main file
> #include 'file'
> to work both for wml started in subdirectory and for
> wmk started in root directory?

Hi Zbigniew,

write
  #include "file"
in your source file and put
  -I your_subdir
in root/.wmkrc
