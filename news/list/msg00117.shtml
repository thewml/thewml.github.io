Information 

This lecture about Perl was held on 26 March 1998
on the total gesamtmeeting of sd&m / TI in Munich.

Documents 

The foil set of the lecture and the Handouts, which
were output to the listeners, are on-line
available:

    Foil record (PowerPoint97) That is the complete foil record by 53
    single foils in the PowerPoint97-Format, i.e. as it was created.
    The file to the Download has 2.1MB. It can be recalled however
    only with explicit acces password!

    Foil record (HTML) That is the complete foil record of 53 single
    foils in the HTML format (naturally generate with the Website META
    Languagea). These pages are particularly suitable for on-line
    second selections.

    Foil record (Postscript, gzip) That is the complete foil record of
    53 single foils in the Postscript format. Each foil is contained
    in originalgroesse in the DIN A4 landscape format. That is the
    collecting main for the foils themselves.  The file to the
    Download has 2.5MB and unpacked 5.7MB.

    Script (Postscript, gzip) That is the Script of the foil record in
    the Postscript format. In each case 2 foils are summarized on DIN
    A4 page in the portrait format. It is particularly suitable for
    later off-line second selections. The file to the Download has
    2.1MB and unpacked 5.7MB.

    Handout (Postscript, gzip) That is the Handout of the foil record
    in the Postscript format. In each case 6 foils are summarized on
    DIN A4 page in the portrait format. It is particularly suitable
    for the compact overview. This format was output to the
    listeners. The file to the Download has 2.1MB and unpacked 5.7MB.
