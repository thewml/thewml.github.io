[ANNOUNCE] WML-1.7.2

  • From: Denis Barbier <nospam@thanx>
  • Date: Mon, 28 Jun 1999 19:12:09 +0200 (CET)
Dear WML users,

WML-1.7.2 is out.

This release does not reflect the current wml-SNAP, it contains only
bug fixes found at
   http://www.engelschall.com/sw/wml/distrib/bugs/wml-1.7.1/

I believe this version is really stable and WML-1.7.3 should arrive in
September if no major bugs are found.

To upgrade, you can
- download and compile wml-1.7.2.tar.gz (1810Kb)
- use wmu to upgrade
- download and apply wml-1.7.1-1.7.2.sh.gz (3.5Kb)

Yours,

Denis
