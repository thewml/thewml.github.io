Mime-Types and WML source files

  • From: Fritz Zaucker <nospam@thanx>
  • Date: 07 Sep 1999 11:33:34 +0200

Hi,

today I received a request to add the following Mime Types to our
Web-Server configuration:

text/vnd.wap.wml	wml
text/vnd.wap.wmlscript	wmlscript wmls
image/wbmp			wbmp

There seems to be an evolving standard for wireless applications (see
http://www.wapforum.org).


This was in conflict with our server setting 

AddType text/plain .wml

for displaying WML source files as text/plain.

Any comments (for example by Ralf)? Should the WML community move to a
different filename extension?
