<barbier@imacs.polytechnique.fr> wrote:

> Is there a portable way to set locale to `fr_FR'?

No. You have to do a 'locale -a' to find the local locales and to set LC_* accordingly.

Here is an example on a Tru64 machine:

nefertiti:~ > locale -a |grep -i fr
fr_BE.ISO8859-1
fr_CA.ISO8859-1
fr_CH.ISO8859-1
fr_FR.ISO8859-1
