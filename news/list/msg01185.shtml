I know you're right. Ok, let's go:
    http://imacs.polytechnique.fr/~barbier/mp4h/

Some words of explanation: to use it within WML, you have to
* download a recent version of WML (recent means no more than 1 day)
* compile mp4h and install it in the same directory than wml_p2_mhc,
  rename it to wml_p2_mp4h.
* with the -z flag, wml uses wml_p2_mp4h. Without it, wml_p2_mhc is
  used.

This is still experimental, improvements of any kind are welcome.
