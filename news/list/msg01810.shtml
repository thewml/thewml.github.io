Hi folks,

i receive very positive feedback about mp4h by non-WML users, and some
requests.  One of them is to make mp4h more extendable.

So i first added support for text file inclusion, this was fairly
simple.  Then i remember that development versions of m4 has support
for loadable modules, so i put it back into mp4h.  It looks really
great, it has been tested on SunOS, HPUX, FreeBSD and Linux.

But this work needs a more experienced software designer. Any advice or 
patches would help me a lot.

Current development version of mp4h is
   http://www.engelschall.com/sw/mp4h/distrib/mp4h-1.0a602.tar.gz
There is a sample module in the modules/ directory.
