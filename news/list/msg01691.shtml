For those who want to give it a try, i will put some mp4h snapshots at
   http://www.engelschall.com/sw/mp4h/distrib/

Tonight release is mp4h-0.9a601.tar.gz
It implements the different choices given in the previous post with the
--expansion=NUMBER command line option, where NUMBER is a bit sum of

  1  do not parse unknown tags
  2  unknown tags are assumed being complex
  4  an unknown tag whose name ends with a star is assumed simple

Roughly speaking,
  case A => 1
  case B => 0
  case C => 6

On the other hand, i played with quoting mechanism, and it is awfully
broken in this release.
