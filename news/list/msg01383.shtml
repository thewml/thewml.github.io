Hello,

There seems to be some confusion of terms here. The WML I am familiar with
is the Wireless Markup Language, an SGML derivative optimized for developing
internet content for narrowband handheld wireless devices (primarily
phones). The WAP applications you speak of above refer to the Wireless
Applications Protocol. The WML discussed on this list is not the same WML
used to create WAP applications.
I'm pretty sure The WAP Forum has a patent on the name WML as well so
perhaps the owner of this list should be advised of that.
I work for Phone.com, we are the WAP founder and leader in the industry.
Check out www.phone.com for more information on developing WAP applications.
There is a free downloadable SDK available from their developer website at
http://updev.phone.com/

I hope this is helpful
Regards,
David J. Keith
