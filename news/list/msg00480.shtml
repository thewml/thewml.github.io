Sliced, folded navbars

  • From: Florian Hars <nospam@thanx>
  • Date: 14 Mar 1999 21:45:43 +0100
I somehow tweaked my code for hierarchical navbars to produce slices
for different languages. So if you always wanted to do things like 

  <navbar:button id=www.wml.navbars
  url="www/wml_navbars.#.html|www/wml_navbars.html"
  txt="<de:Navigationsleisten><en: Navigation&nbsp;Bars>">

you might want to take a look at http://www.hars.de/www/wml_navbars.html
resp. http://www.hars.de/www/wml_navbars.en.html.

I managed to rebuild my site with these definitions (and three pages
even use the slice functionality), so it can't be totally broken.

The documentation could probably be better, but if you understand
wml::des::navbar, you should be able to make some sense of it, my code
is only a modification of it and should almost work as a drop in
replacement. And, yes, the resulting slices have [LANG_XX:<img src=...>:]
instead of the slightly easier to achieve <img src=[LANG_XX:...:]>,
so the beautifier will actually insert the image dimensions.

Yours, Florian.
