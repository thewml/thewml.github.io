height and width trough rollover

  • From: Frederic Le Mouel <nospam@thanx>

Hi,

When I pass height and width to rollover tag, height and width
are well-passed to html img tag, but when I use lang tag in 
rollover tag, height and width arguments don't passed.

Ex:

Wml:

<rollover name    = "<lang:en: Grompf><lang:fr: Grumpf>"
	  height  = 10
	  width   = 5
	  src     = <lang:star: ./Icons/pic-*-off.gif>
	  oversrc = <lang:star: ./Icons/pic-*-on.gif>
	  href    = <lang:star: ./page.html.*>
	  alt     = "<lang:en: Arf><lang:fr: Urf>">

Html.fr, idem html.en:

<a href="./page.html.fr"
   onMouseOver="ro_imgOver('ro_img_Grumpf', 'Urf'); return true"
   onMouseOut="ro_imgNormal('ro_img_Grumpf'); return true"
><img
   name="ro_img_Grumpf"
   src="./Icons/pic-fr-off.gif"
   alt="Urf"
   border="0"
></a>
