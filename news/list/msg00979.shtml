WML 1.7.2: Undocumented feature in wml::des::navbar

  • From: Hans Ulrich Niedermann <nospam@thanx>
  • Date: 20 Sep 1999 15:15:09 +0200
Hi all,

in wml::des::navbar(3) a description of the "nohints" param to the
<navbar:render> tag is missing. My suggestion is to add 

       nohints
           Do not create Javascript hints for navigation buttons

or something similar. The current text is:

       Options of <navbar:render>:

       name=STR
           The name of the navigation bar definition to use when
           rendering.

       select=ID
           Select a particular button as selected.

       subselected
           Marks the selected button as a subselected one, i.e.
           the current page for which the button is selected is
           deeper than the original page for which this button
           stands.

       txtonly
           Forces the rendering to ignore all defined images.

Greetings,
