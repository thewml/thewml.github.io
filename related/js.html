JavaScript

These are references to JavaScript, Netscape's client-side scripting language. Current version is 1.2, supported in Netscape Communicator 4.0. The previous version 1.1 is supported in Netscape Communicator 3.0 and Internet Explorer 4. JavaScript code snippets can be easily included into pages by using WML's <javascript> container tag from wml::std::tags.
