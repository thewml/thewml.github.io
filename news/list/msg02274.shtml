> 
> On 27-Jun-01 Stephan Petersen wrote:
> > <meta name="MSSmartTagsPreventParsing" content="TRUE">
> 
> Thanks a lot! How do you insert it with <info> ?
> (I modified the official (1.74) code of the info-tag to allow
> it).

This is not related to <info>, which only displays informatin about
source document and generator.
But you may add this line in your template file along with
<info style=meta>, or if you use wml::std::page add
  <head>
    <meta name="MSSmartTagsPreventParsing" content="TRUE">
  </head>
so that this meta tag appears on each page.

If you want this tag to be automatically inserted by wml::std::page,
please send a piece of text i could include in documentation.
