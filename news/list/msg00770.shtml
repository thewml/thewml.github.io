BUG REPORT

Package: WML 1.7.1
Operating System: Red Hat Linux 5.2 (kernel 2.2.9)

Problem Description:
| Very, very minor problem:
| 
| <href url=$(IGNORE_THIS) hint="John's Possessive">
| 
| yields:
| 
| <a href=$(IGNORE_THIS)
| onMouseOver="self.status='John's Possessive';return true">
| 
| Causing the onMouseOver to fail.  It needs to create:
| 
| <a href=$(IGNORE_THIS)
| onMouseOver="self.status='John\'s Possessive';return true">
| 
| to work properly.
| 
| Current Workaround:
| 
| <href url=$(IGNORE_THIS) hint="John\\'s Possessive">
|
