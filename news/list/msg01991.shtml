  This is not a serious problem.
"<case:upper></PRE></case:upper>" isn't upper on wml 2.0.4.
wml 2.0.0 is OK.

Built Environment:
    Host: i586-pc-linux-gnu2
    Perl: 5.005 (/usr/bin/perl) jperl5.005_03-990822 EUC version

---- << input 
#use wml::std::case
<case:upper>ORIGINAL<pre>LOWER</pre><PRE>UPPER</PRE></case:upper>
<case:lower>original<pre>lower</pre><PRE>upper</PRE></case:lower>

---- >> output
ORIGINAL<PRE>LOWER</pre><PRE>UPPER</pre>
original<pre>lower</pre><pre>upper</pre>
