SGML-formatting with WML

  • From: nospam@thanx ("Dirk Frömbgen")
  • Date: Wed, 8 Dec 1999 13:55:08 +0100
Has anyone ever thought of using WML as a batch formatter for
SGML input?

Benefits:
This would allow WML to be used with structured editors.
User profiles (i.e. customized subsets) of WML could be created,
for specific purposes, hiding as much technical detail as
possible.

This would be just one more language on top of WML.

Next year, I might be in a position to work on this.
I'd prefer a canned solution.
