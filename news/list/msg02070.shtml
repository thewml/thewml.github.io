On Wed, Nov 22, 2000 at 07:48:28PM +0100, Thomas R. Koll wrote:
> Hi,
> I don't know much about settip up emacs, so I want to know where to install the files, I tried /usr/share/emacs/20.7/lisp but it didn't work.
> (My system is debian)
> I hope you can help me
> 
I believe the location you want is /usr/local/share/emacs/site-lisp.
You don't want to use /usr/share/emacs because that is the domain
of the package and the file you added could be removed during an
upgrade. Only changes to configuration files are guaranteed to be
safe in the system area.

Another response talked about the validity of the lisp given.
